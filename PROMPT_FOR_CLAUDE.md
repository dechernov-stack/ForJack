# Готовый промпт для Claude Code

## Как пользоваться (для тебя, аналитик)

1. Перенеси папку `storytelling-bot/` на свою машину (rsync / git push / архив — на твой вкус).
2. Открой её в **Claude Code** (`cd storytelling-bot && claude`).
3. Положи рядом `DE.pdf` (исходный фреймворк) — он понадобится Claude Code для few-shot примеров.
4. Скопируй блок ниже целиком в чат с Claude Code и отправь.
5. Дальше Claude задаст вопросы по `.env` и попросит ключи. Дай их или скажи «используй mock» для тех, что пока нет.

## ⬇⬇⬇ Копировать всё ниже в Claude Code ⬇⬇⬇

```
Привет. Прочитай CLAUDE_TASKS.md в этой папке — там пошаговый план превращения
прототипа storytelling_bot.py в production-репо. Также прочитай architecture.md
и README.md для контекста. Файл DE.pdf рядом — это исходный фреймворк сторителлинга,
используй его для few-shot примеров классификации.

Поведение:
- Выполняй задачи строго по порядку (Task 1 → Task 2 → … → Task 12).
- После каждой завершённой задачи делай git commit с сообщением вида
  "<task-N>: <короткий subject>".
- Перед началом каждой задачи перечитывай её Definition of Done (DoD)
  и убедись, что всё условия достижимы.
- Если задача требует API-ключа, которого у меня нет — предложи альтернативу
  (mock или freemium-источник) и спроси согласие, прежде чем переключаться.
- Никогда не нарушай два правила: provenance 100% (каждый факт со ссылкой)
  и human-in-the-loop для decision="terminate".
- На каждом шаге, где трогаешь LLM-вызовы, оборачивай в langfuse-трейсинг,
  если LANGFUSE_PUBLIC_KEY задан в .env.

Целевая компания для финального прогона (Task 10):
у меня есть валидные ключи Tavily и Anthropic — проведи end-to-end по компании
**Stripe** (или, если попросит пользователь, Anthropic). На Theranos прогонять
отдельно как regression-test для red-flags.

Перед стартом задай ОДИН блок уточняющих вопросов:
- какие из API-ключей есть (Anthropic / Tavily / Crunchbase / GitHub)?
- какой LLM-провайдер по умолчанию?
- куда пушить репо (GitHub URL или local-only)?
- целевая компания для Task 10 (Stripe / Anthropic / другое)?

После ответа — стартуй с Task 1 и не останавливайся, пока не дойдёшь до Task 12
или не упрёшься в блокер. Используй TodoWrite для трекинга прогресса.

Финальный артефакт — публичный/приватный git-репо + reports/<entity>.html,
который я смогу открыть в браузере.
```

## Что делать, если что-то сломалось

| Симптом                                                        | Что попросить у Claude Code                                              |
|----------------------------------------------------------------|--------------------------------------------------------------------------|
| Падает на `pytest` после Task 2                                | «Откати Task 2 в локальный коммит и попробуй другую декомпозицию»        |
| Tavily-rate-limit                                              | «Переключись на Brave Search через httpx»                                |
| Claude API вылет на длинном промпте                            | «Пересмотри chunking в `synthesize_layer`, лимит 8K токенов на вызов»    |
| Whisper падает по памяти                                       | «Используй `tiny` модель и `WHISPER_DEVICE=cpu`»                          |
| Decision выходит странный                                      | «Распиши rationale пошагово в логах, покажи мне, где сработало правило»  |
| Coverage <30% после Task 10                                    | «Подними `Tavily` лимит до 30 результатов, добавь Wikipedia в Research»  |
